## A collection of diagnostic and reverse engineering tools we made to handle XML-based flash files, like those of MLP FiM
---
The most useful script you may find is fla_export_gui.py \
run it with ' python fla_export_gui.py ' \
This will open a browser pointing to 127.0.0.1:7272, this tool will let you explore eveything fla_inspect.py can find inside your .fla \
Note only moder CS5+ .fla files work. Earlier Flash files were encoded in binary form and the spec for those were never released... we might develop exporters later as a fun quest, but that's outside the scope of Flare's current goal.
